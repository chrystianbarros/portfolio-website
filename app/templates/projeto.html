Sobre este projeto

O objetivo deste projeto foi desenvolver um website para hospedagem do meu portfólio profissional. Para fazê-lo, utilizei as tecnologias representadas pelos ícones acima, sendo elas, respectivamente:

  • Bootstrap: utilização de templates HTML e CSS para construção visual do website;
  • Docker: desenvolvimento de uma imagem de container contendo todas as dependências necessárias para execução da aplicação;
  • Flask: desenvolvimento da aplicação web por meio da linguagem Python;
  • Gunicorn: hospedagem da aplicação web em um WSGI HTTP Server;
  • HTML: manipulação da estrutura e textos do website;
  • Linux: hospedagem da aplicação e containers em um servidor Linux Ubuntu;
  • NGINX: utilização de um web server como proxy-reverso, para balanceamento de carga entre containers;
  • Oracle Cloud Infrastructure: provisionamento de máquina virtual Linux, hospedagem e publicação da aplicação;
  • Python: lingaugem utilizada para criação da aplicação web e back-end.
